Phillips Curve OOS Forecasting (Post-Pandemic)

TL;DR: Tested whether Phillips Curve specifications could forecast US inflation out-of-sample after COVID, compared against simple benchmark models (train through 2019Q4; test 2020Q1–2023Q3).

Problem

Post-pandemic inflation was persistently underpredicted by many standard models. This project evaluates whether a Phillips Curve relationship (inflation expectations + slack) retained forecasting power in the 2020–2023 period, and what additions improve performance.

Data

  • Quarterly US macro dataset (1968Q1–2023Q3).
  • Train/test split: train ends 2019Q4; test window 2020Q1–2023Q3.
  • Target: PCE inflation (headline; core used for robustness). Inputs include expectations proxies and slack measures, with optional cost-push proxies.

Approach

  • Benchmarks: no-change / persistence baselines, AR(1), trailing mean.
  • Phillips Curve variants: expectations proxy sensitivity + alternative slack measures.
  • Extensions: add cost-push proxies (e.g., energy/import prices) to test whether omitted supply shocks explain the post-pandemic miss.
  • Evaluation: pseudo out-of-sample forecasting; RMSE/MAE/Bias over the test window.

Results

Baseline Phillips Curve variants tend to underpredict the 2021–22 inflation surge relative to simple persistence benchmarks. Adding cost-push proxies improves performance for headline inflation in the COVID window.
